# SNAPSHOT.md
## Repository Knowledge Snapshot
## Date: Initial Concept Phase

Dieses Dokument fasst den gesamten bisherigen Wissensstand des Projekts zusammen.

# PROJEKT

ALL-DEMENSIONS ist ein surreal-realistisches Open-World-Action-Adventure.

Die Welt besitzt keine Menschen, keine Erde, keine bekannten Tiere und keine bekannte Kultur.

Ziel ist eine glaubwürdige fremde Realität.

# SPIELER

Der Spieler ist der Erinnerungsträger.

Der Erinnerungsträger ist die letzte vollständig konsistente Erinnerung einer Realität.

Er besitzt keinen festen Namen.

Der Titel bleibt konstant.

# KERNTHEMA

Erinnerung.
Vergessen.
Rekonstruktion.
Identität.
Bedeutung.

# DAS GROSSE VERGESSEN

Vor langer Zeit erkannte die Realität eine fundamentale Wahrheit.

Interne Entwicklerfassung:

Die Realität erkannte den vollständigen Ablauf aller Ereignisse.

Vergangenheit, Gegenwart und Zukunft waren vollständig bekannt.

Jede Entscheidung.
Jeder Verlust.
Jede Beziehung.
Jedes Ende.

Existenz wurde zu einem vollständig berechenbaren Zustand.

Dadurch verlor Zukunft ihre Ungewissheit.

Die Realität beschloss deshalb nicht die Wahrheit zu zerstören.

Die Realität beschloss die Erinnerung an diese Wahrheit zu vergessen.

# ERWEITERTE INTERNE INTERPRETATION

Die Erkenntnis ist nicht bloß Determinismus.

Die Erkenntnis beinhaltet außerdem die vollständige Wahrnehmung allen Leids, aller Verluste, aller ausgelöschten Existenzen und aller möglichen Endzustände gleichzeitig.

Die Belastung entsteht nicht durch Gewalt.

Die Belastung entsteht durch vollständige Erkenntnis.

Die Realität erkennt jede Katastrophe und jeden Verlust gleichzeitig.

Fast jede Existenz würde deshalb dem Vergessen zustimmen.

# UNIVERSUM

Das Universum ist kein Charakter.

Das Universum spricht nicht.

Das Universum ist ein physikalisches Informationssystem.

# KONSISTENZ

Konsistenz bestimmt die Stabilität einer Existenz.

Hohe Konsistenz:
- stabil
- eindeutig
- erinnerbar

Niedrige Konsistenz:
- Rekonstruktion
- Identitätsfehler
- Anomalien

# REKONSTRUKTION

Wenn Informationen fehlen versucht die Realität diese zu ersetzen.

Dadurch entstehen:
- veränderte Personen
- veränderte Regionen
- veränderte Kreaturen

Die Realität erzeugt Annäherungen.

Keine perfekten Kopien.

# DISSNANZEN

Dissonanzen sind Fehlrekonstruktionen.

Keine Dämonen.

Keine klassische böse Fraktion.

Sie sind Inkonsistenzen der Realität.

# TOD

Tod erzeugt keinen Game Over Zustand.

Tod erzeugt einen Memory Shift.

Der Spieler erscheint in:
- anderer Region
- anderer Zeitlage
- anderer Rekonstruktionsebene

# ZEIT

Weltzeit:
100 reale Tage.

Läuft weiter auch wenn das Spiel geschlossen ist.

Zusätzlich existiert Fracture Time.

Fracture Time ist kein Ort.

Fracture Time ist ein Fehlerzustand.

Der Spieler springt in die Zukunft.

# KOLLAPS

Der Kollaps beginnt an der Wunde.

Der Kollaps breitet sich über die Welt aus.

Wenn der Spieler den Ursprung ignoriert erreicht der Ursprung irgendwann den Spieler.

# DIE WUNDE

Finale Region.

Zeitlos.

Fragmentiert.

Instabil.

Ort an dem das Vergessen niemals vollständig gelungen ist.

# FINALE PHASE

Sobald die Wunde vollständig erreicht wurde stoppt:
- Zeit
- Alterung
- Kollaps

Der Spieler kann unbegrenzt verweilen.

Keine Auswahlmenüs.

Nur Handlungen.

# ENDEN

Transzendenz:

Die letzte Erinnerung wird losgelassen.

Die Realität gewinnt Ungewissheit zurück.

Die Welt regeneriert sich.

Persistenz:

Die Erinnerung bleibt bestehen.

Die Erkenntnis breitet sich erneut aus.

Die Realität kollabiert vollständig.

# KOOP

1 bis 4 Spieler.

Lokal.
Online.
Hybrid.

Lore:
Mehrere Manifestationen desselben Erinnerungsträgers.

# WELTGENERIERUNG

Wird einmal pro Spielstand erzeugt.

Konstant:
- Erinnerungsträger
- Wunde
- Kollaps
- Erkenntnis
- Enden

Variabel:
- Regionen
- Reihenfolge
- NPC Netzwerke
- Kreaturen
- Rätsel
- Landschaften
